# React world Example : Chalk

এই অধ্যায়ে আমরা Chalk.js এর মত একটা ছোট লাইব্রেরি বানাব যেটার মাধ্যমে কনসোলে যেকোন কিছু style করে প্রিন্ট করতে পারব ।
প্রথমত আমরা রিকুয়ারমেন্টগুলো একটু দেখি ।


```tsx
class Chalk {
  private text: string;

constructor(text: string = "") {
this.text = text;
}

red(): this {
this.text = "\x1b[31m" + this.text + "\x1b[0m";
return this;
}

bgBlack(): this {
this.text = "\x1b[40m" + this.text + "\x1b[0m";
return this;
}

underline(): this {
this.text = "\x1b[4m" + this.text + "\x1b[0m";
return this;
}

bold(): this {
this.text = "\x1b[1m" + this.text + "\x1b[0m";
return this;
}

value(): string {
return this.text;
}
}

export const chalk = new Chalk("hello");

```

#### এক্ষেত্রে এটাকে ব্যবহার করতে পারি যেভাবেঃ 

```tsx 

// Create a new instance with some text
const chalk = new Chalk("Hello World");

// Basic usage with a single style
console.log(chalk.red().value()); // Red text
console.log(chalk.bold().value()); // Bold text

// Chaining multiple styles
console.log(
chalk
.red()
.bold()
.underline()
.value()
); // Red, bold and underlined text

// Create new instance for different text
const warning = new Chalk("Warning!");
console.log(
warning
.red()
.bgBlack()
.bold()
.value()
); // Red text with black background and bold

// You can also create an instance with empty text
const emptyChalk = new Chalk();
// Then use it with different strings
console.log(new Chalk("Success!").bold().value());

```
