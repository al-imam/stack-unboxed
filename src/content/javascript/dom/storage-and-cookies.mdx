# Web Storage

Web Storage অনেক প্রকার আছে যেমন।

1.  LocalStorage
2.  SessionStorage
3.  SharedStorage (Experimental)
4.  Cookie
5.  IndexedDB
6.  WebSQL.

এখন এগুলা বিস্তারিত বিশ্লেষণ করা হল । 

## Local storage

Local storage হলো একটি ক্লায়েন্ট সাইড storage যেখানে কোনো DOM এর উপাদান নিয়ে কাজ
করা হয় না ।

## session storage

Session storage হলো একটি ক্লায়েন্ট সাইড storage যেখানে কোনো DOM এর উপাদান নিয়ে
কাজ করা হয় না ।
